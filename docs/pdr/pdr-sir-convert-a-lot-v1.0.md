# Sir Convert-a-Lot Product Description v1.0

## High-Level Value

Sir Convert-a-Lot provides one canonical conversion surface for developers and assistants,
turning heterogeneous document conversion workflows into deterministic, auditable, and
Hemma-offloadable operations.

## Feature Themes

- One CLI/API surface for conversion jobs.
- GPU-first offloading path on Hemma for heavy workloads.
- Deterministic manifests and error semantics for LLM-safe automation.
- Cross-repo integration contract for HuleEdu and Skriptoteket.
